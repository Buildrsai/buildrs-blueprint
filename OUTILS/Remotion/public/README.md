# Assets audio — Buildrs Blueprint Ad

Place ici les fichiers audio avant de rendre la vidéo :

## music.mp3
Musique de fond dark/cinématique.
- Volume mixé à -15dB automatiquement dans le code (volume={0.18})
- Durée recommandée : 45s minimum
- Tempo moyen, build progressif

## voiceover.mp3
Voix off complète (40 secondes).
- Volume plein (volume={1})
- Script dans le design doc : docs/superpowers/specs/2026-03-23-buildrs-blueprint-ad-design.md

> Si les fichiers sont absents, Remotion ignorera silencieusement les tags <Audio>.
> Pour le studio, vous pouvez tester sans ces fichiers.
