Buildrs Lab — Design V1.5

3 approches pour se rapprocher de l'esthétique Google Antigravity. Clique pour sélectionner celle qui te parle.

A

Particules & Curseur

Changements ciblés, impact max. Remplace les dots ronds par des mini-rectangles (comme Antigravity — 2×7px, rotation légère, multicolore bleu). Ajoute un curseur clignotant sur le H1 hero (bleu #3279F9). Section pricing : dot grid statique en fond (grille de points gris réguliers). Rien d'autre ne change — juste ces 3 éléments précis qui font toute la différence.

🎯 Effort : 1-2h · Impact visuel : +30% · Risque : Zéro
B

Antigravity Complet ⭐

L'expérience Antigravity traduite dans Buildrs. Tout de A, plus : scroll-reveal sur tous les éléments (fade + translateY, 300ms), horizontal tool strip entre sections (logos outils qui scrollent en boucle — Claude, Supabase, Vercel…), hero headline left-align sur les sections features (comme Antigravity), sections light/dark avec transition franche encore plus marquée, et hover states cinematiques sur les cards (scale + glow subtil).

🚀 Effort : 4-6h · Impact visuel : +70% · Risque : Faible — recommandé
C

Zero-Gravity Cinematic

Le niveau de finition ultime, pour se démarquer vraiment. Tout de B, plus : parallax sur le hero (particules + headline bougent à des vitesses différentes au scroll), floating product mockup dans le hero (screenshot du Lab en lévitation, rotation légère), background mesh gradient très subtil derrière le hero blanc (bleu/lavande quasi invisible), et number counter animation sur les stats (ex: "87 projets lancés" qui compte).

✨ Effort : 8-12h · Impact visuel : +100% · Risque : Moyen — à faire après backend
